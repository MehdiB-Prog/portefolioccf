Projet 5 : Application NetBouquet

📌 Contexte

NetBouquet est une entreprise spécialisée dans la vente de bouquets et de plantes. Les commandes étaient initialement réalisées par téléphone ou par mail. L’objectif du projet était de moderniser ce système en développant un site web.

🎯 Objectifs

Créer une application web permettant aux clients de consulter les produits et de passer des commandes en ligne, tout en offrant une interface d’administration.

👥 Utilisateurs

  • Client : consulter les produits, passer commande, voir ses commandes
  • Admin : gérer les bouquets (ajout, modification, suppression)

⚙️ Fonctionnalités

  • Système de connexion sécurisé
  • CRUD complet (INSERT, UPDATE, DELETE)
  • Affichage des bouquets
  • Gestion des commandes
  • Interface client et interface administrateur

💻 Technologies utilisées

  • PHP
  • MySQL
  • HTML / CSS
  • Architecture MVC
  • UwAmp (serveur local)

🏗️ Architecture

Le projet est basé sur une architecture MVC (Modèle - Vue - Contrôleur), permettant de séparer la logique métier, l’affichage et les accès aux données.

🔐 Sécurité

  • Hashage des mots de passe
  • Protection contre les injections SQL
  • Gestion des accès utilisateurs

🔄 Réalisation

  • Analyse du cahier des charges
  • Installation de la base de données
  • Développement de la partie publique
  • Développement de la partie admin
  • Gestion des utilisateurs
  • Tests fonctionnels

🧪 Tests

Des tests ont été réalisés pour vérifier le bon fonctionnement des fonctionnalités, notamment les opérations CRUD et la connexion utilisateur.

⚠️ Difficultés rencontrées

  • Mise en place de l’architecture MVC
  • Gestion des sessions utilisateurs
  • Sécurisation du site

🧠 Compétences développées

  • Développement web complet
  • Création d’un CRUD
  • Gestion d’une base de données
  • Sécurisation d’une application
  • Organisation du code

📚 Conclusion

Ce projet m’a permis de développer une application web complète, en mettant en pratique mes compétences en PHP, SQL et architecture MVC. Il représente une étape importante dans mon apprentissage du développement.
