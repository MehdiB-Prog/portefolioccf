📱 Projet Pokedex

📌 Présentation

Le projet Pokedex est une application web développée dans le cadre de ma formation en BTS SIO. Il s'agit d'une application dynamique permettant de gérer une base de données de Pokémon.

Ce projet m’a permis de mettre en pratique le développement web côté serveur ainsi que la gestion de base de données.

⚙️ Fonctionnalités

  • 🔐 Système de gestion des données (CRUD)
  • ➕ Ajout de Pokémon
  • ✏️ Modification des informations
  • ❌ Suppression de Pokémon
  • 📋 Affichage des Pokémon sous forme de tableau
  • 🔎 Accès aux détails d’un Pokémon

💻 Technologies utilisées

  • HTML / CSS
  • PHP
  • MySQL
  • UwAmp (serveur local)

🎯 Compétences développées

  • Gestion d’une base de données relationnelle
  • Création d’un CRUD complet
  • Utilisation de requêtes SQL (INSERT, UPDATE, DELETE)
  • Sécurisation des données (injections SQL, gestion des formulaires)
  • Structuration d’une application web

📸 Captures d’écran

⚠️ Hébergement

Ce projet utilise PHP et une base de données MySQL. Il est donc fonctionnel uniquement en local via UwAmp (ou XAMPP). GitHub Pages ne supporte que les sites statiques (HTML, CSS, JavaScript), ce qui empêche son hébergement en ligne.
