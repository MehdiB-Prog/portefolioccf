Projet Arboretum (Java)

📌 Contexte

L’Arboretum de Versailles-Chèvreloup est un parc regroupant de nombreuses espèces d’arbres provenant du monde entier. Dans le cadre de ce projet, l’objectif était de développer une application permettant de gérer les visites et les informations liées à cet arboretum.

🎯 Objectifs

Le projet avait pour objectif de créer une application capable de gérer les visites, d’exploiter une base de données existante et de fournir des informations utiles aux gestionnaires (statistiques, données, etc.).

👥 Utilisateurs

L’application est destinée aux gestionnaires de l’arboretum afin de faciliter la gestion des données et des visites.

⚙️ Fonctionnalités

  • Menu interactif avec plusieurs options
  • Affichage des données
  • Recherche d’informations
  • Interaction utilisateur via console
  • Manipulation de données issues d’une base MySQL

💻 Technologies utilisées

  • Java (application console)
  • MySQL (base de données)
  • NetBeans
  • GitHub

🏗️ Architecture

Le projet est structuré en architecture en couches, permettant de séparer les différentes responsabilités (accès aux données, logique métier, affichage).

🔄 Réalisation

  • Lecture et analyse du cahier des charges
  • Développement des fonctionnalités en Java
  • Connexion à la base de données
  • Tests fonctionnels
  • Réalisation de tests unitaires
  • Documentation avec Javadoc

🧪 Tests

Des tests ont été réalisés pour vérifier le bon fonctionnement de l’application, notamment les interactions avec la base de données et les choix du menu.

⚠️ Contraintes

  • Respect des normes de codage Java
  • Architecture en couches
  • Utilisation d’une base de données existante

🧠 Compétences développées

  • Programmation en Java
  • Connexion à une base de données
  • Organisation d’un projet
  • Tests et validation
  • Documentation technique

📚 Conclusion

Ce projet m’a permis de renforcer mes compétences en développement Java, de mieux comprendre la gestion d’une base de données et de structurer une application complète avec un menu interactif.
